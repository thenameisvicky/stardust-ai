StardustAI

Iinference engine for AI-powered SaaS | Streaming | Queue | Self hosted | Cloud hosted

Get stardust on your machine
bash
docker pull vickystardust5/stardust:latest
Install Dependencies
bash
docker run -d --name rabbitmq -p 5672:5672 rabbitmq:3-management
docker run -d --name ollama -p 11434:11434 ollama/ollama
Spin up Stardust
bash
docker run -it --rm --network="host" --memory="256m" -e AMQP_HOST=127.0.0.1 -e OLLAMA_HOST=http://127.0.0.1:11434 vickystardust5/stardust:latest
